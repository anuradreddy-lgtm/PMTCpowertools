# PMTCpowertools 🛠️

PMTCpowertools is a modern web application designed to optimize shop management and increase sales efficiency. It provides a streamlined interface for monitoring shop operations, managing data, and ensuring a seamless user experience.

---

### 🚀 Features
* **Modern UI:** Responsive design with support for both Dark and Light modes.
* **Optimized Performance:** Built for speed and reliability in a retail environment.
* **Secure Access:** Implements Row-Level Security (RLS) to ensure data integrity.
* **Live Deployment:** Fully integrated CI/CD pipeline for instant updates.
